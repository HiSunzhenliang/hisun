LeetCode-242.Valid-Anagram

Given two strings s and t , write a function to determine if t is an anagram of s.

解题思路

长度不同 返回false
一遍扫描 Hash 计数，s 出现字母 ++，t 出现字母 –，最后判断是否全为 0 即可
