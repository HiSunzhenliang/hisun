Python getopt简单使用

Python getopt的简单使用

一般 getopt使用在解析参数列表上。

格式

一般情况下，选项有“短选项”和“长选项”两种

短选项格式：一个减号一个字母；长选项格式：俩减号多个字母

getopt.getopt 方法有仨参数：要处理的对象列表、短选项组、长选项组

  • 短选项组为字符串，若选项有参数，后面加冒号
  • 长选项组为列表，若选项有参数，后面加等号

该方法返回值为二元元组，元组中每个元素都是列表, 一个是选项解析结果，另一个是其余参数
